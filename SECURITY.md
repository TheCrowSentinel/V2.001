# SECURITY.md

## Supported release
Crow v2.001 is a public XRPL Testnet development prototype.

## Never report secrets
Do not submit wallet seeds, private keys, recovery phrases, authentication tokens,
or personal financial information in a public issue.

## Prototype limitations
- Testnet only.
- No production custody.
- No production identity verification.
- No Mainnet support.
- No guarantee of availability.
- No formal dispute-resolution service.

## Reporting
Use a private security contact before a public launch. Add that contact here only
after a monitored security mailbox exists.
